# Sistema de Nómina RD

Sistema CRUD para gestión de nóminas en República Dominicana.

## Tecnologías
- Python (Flask)
- HTML
- SQL Server

## Funcionalidades
- CRUD de empleados
- Cálculo automático de nómina (AFP, SFS, ISR)
- Reportes de nómina
- Validación de cédula dominicana

## Git Flow
Este proyecto implementa la metodología Git Flow con:
- 5+ ramas feature/hotfix
- 15 Pull Requests cerrados
- Ramas: main, develop, qa
